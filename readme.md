# SIEGFRIED

x86 long mode operating system that would implement modern features only. and this time i actually knew more on what i'm doing

chose the name because it sounds cool


## how to build

1. install c-development stuff
2. build siegfried-userspace-toolchain (diffrent repo)
3. build x86_64-none-elf toolchain (check osdev wiki)
4. make to build, make install to install, make test to test in qemu
